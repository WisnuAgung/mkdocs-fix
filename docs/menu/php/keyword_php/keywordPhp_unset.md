#**Keyword PHP Unset**
***

## **A. Penjelasan**

Merupakan kebalikan dari isset(), yaitu untuk meng-unsetvariabel,dan membebaskan memory yang sudah digunakan dan juga berguna untuk menghapus sebuah variable
 
## **B. Bentuk Syntax Umum**

	unset($variabel);
       
## **C. Implementasi**

	<?php
	
		$nama="Fatma Zahrotun Nisak";
		
		unset($nama);
		
		echo $nama;
	?>
