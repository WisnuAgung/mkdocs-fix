#**Keyword PHP IF**
***

## **A. Penjelasan**

If digunakan untuk mengeksekusi beberapa kode hanya jika kondisi tertentu adalah benar.
 

## **B. Bentuk Syntax Umum**

	if(syarat) {
	statement
	}

## **C. Implementasi**

	<?php
		$hasilpenjumlahan=1+2;
		if($hasilpenjumlahan==3) {
		echo "hasil benar";
		}
		?>
